✨ What This Template Gives You

A production-grade monorepo with agentic development standards, AI-powered code review, and automated CI/CD — ready to go.

🤖

AI Code Review

CrewAI-powered multi-agent review system runs on every PR. Analyzes diffs, detects issues, and posts findings to GitHub Actions. Learns from your feedback over time.

🔄

CI/CD Pipeline

Phase-based GitHub Actions: format & lint (auto-fix), test, build, deploy preview, deploy production, AI review. All reusable workflows.

📋

Agentic Framework

Complete agent operating system: AGENTS.md entry point, style guides, templates, ADRs, autonomy boundaries. Agents read these and follow your standards.

📦

Everything is Code

PRs, issues, and kanban boards managed as markdown files in docs/. Version-controlled, searchable, portable. GitHub is the comment layer, not the source of truth.

🧠

Review Memory

The AI review crew remembers your preferences across runs. Tell it "stop flagging X" and it won't flag it again. JSON-in-repo by default, mem0 Cloud optional.

🚀

Cloudflare Deploy

Preview deploys on labeled PRs, production deploy on merge to main. Cloudflare Pages with custom domains. Free tier covers most projects.
